== Executing the Integration Tests with the SSH Test Server

The test server image exposes the SSH port 22 which is mapped to 2222 and the 
current directory is mapped under /app in the docker container:

[source, bash]
----

# building and running the docker image
docker build -t ssh_test_image test-ssh-server
docker run -d -v "$($pwd):/app" -p 2222:22 --name ssh_test ssh_test_image

# running the unit tests and integration tests
dotnet test src

# stopping and removing the docker image
docker stop ssh_test
docker rm ssh_test
----
